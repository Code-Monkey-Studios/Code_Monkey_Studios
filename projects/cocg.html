Cyberpunk: Online Character Generator {

Hi and welcome to the Cyberpunk: Online Character Generator project. The intent of this project, which you can see is ongoing, was to develop an interactive site that would allow a person with limited knowledge of the role-playing game to select the books his/her referee has designated as available and create a character and print it without the referee's express supervision. The project uses jQuery plug-ins for a lot of the interactive elements but the actual character creation is coded entirely in javascript.
